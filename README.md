# Boston Housing Price Prediction System

## Overview
The Boston Housing Price Prediction System is a machine learning-based web application developed using Python. The project predicts housing prices based on different input features such as crime rate, number of rooms, property tax, accessibility, and other housing-related factors.

This project demonstrates the implementation of machine learning algorithms, data preprocessing, model training, and deployment through a web interface.

---

## Objectives
- To analyze housing data using machine learning techniques.
- To predict house prices accurately based on user input.
- To build a user-friendly web application for prediction.
- To understand data preprocessing and model evaluation methods.
